📝 Sleep Screen Text for CrossPoint Devices

Create a sleep screen for your X3 or X4 CrossPoint device with useful text!

Note: Due to changes to how the webserver handles CORS, currently requires the nightly build (7/16 or later) to work.

To upload directly to device, put device in File Transfer mode (join the same network or create a hotspot).

Alternatively, download the file and place it manually in the root of the SD card (make sure the file is named sleep.bmp).
